Checklist para Desenvolvimento Web

O que é o Checklist?
Todos nós, desenvolvedores, sabemos o quanto é complexo criar um software de qualidade, ainda mais com a quantidade enorme de plataformas, dispositivos e browsers.
O Checklist é apenas uma lista de itens que devem ser observados antes de publicar qualquer trabalho feito para internet.
Essa ferramenta auxilia e evita dores de cabeça com problemas e erros bobos que poderiam ter sido evitados.

Espero que seja muito útil!
